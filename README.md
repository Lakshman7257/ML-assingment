#  PCA vs Autoencoder on Fashion-MNIST

##  Overview
This project compares **Principal Component Analysis (PCA)** and **Autoencoders** for dimensionality reduction using the Fashion-MNIST dataset.

The aim is to demonstrate how **linear vs non-linear techniques** perform when compressing and reconstructing high-dimensional image data.

---

##  Objectives
- Understand dimensionality reduction techniques  
- Compare PCA (linear) vs Autoencoder (non-linear)  
- Visualize compressed representations  
- Evaluate reconstruction quality using error metrics
