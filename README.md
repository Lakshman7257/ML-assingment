# Label Smoothing vs Standard Training on MNIST

## Overview
This project compares **standard cross-entropy training** and **label smoothing** for training neural networks on the MNIST dataset.  

The aim is to demonstrate how **regularization via label smoothing** affects model confidence, generalization, and prediction calibration compared to standard training.

---

## Objectives
- Understand **label smoothing** as a regularization technique  
- Compare standard training (one-hot targets) vs label-smoothed training  
- Visualize confidence distributions and training loss  
- Evaluate reconstruction and prediction quality using accuracy and confidence metrics
